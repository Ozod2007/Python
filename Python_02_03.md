# Задания с использованием цикла FOR. 
## Задание 1
Рассматривается множество целых чисел, принадлежащих отрезку [1012; 9638], которые делятся на 7 и 
оканчиваются на. Найдите сумму таких чисел.

```
a = 0
for i in range(1012, 9639):
    if (i % 7 == 0) and (i % 10 == 1):
        a += i
print(a)
```
## Задание 2
Рассматривается множество целых чисел, принадлежащих отрезку [1125; 7856], которые делятся а 3 и 
не делятся на 7 и 5. Найдите количество таких чисел. 

```
a = 0
for i in range(1125, 7856):
    if (i % 3 == 0) and not(i % 7 == 0) and not(i % 5 == 0):
        a += 1
print(a)
```
## Задание 3
Рассматривается множество целых чисел, принадлежащих отрезку [100; 999], которые являются 
палиндромом (одинаково читаются как слева направо, так и справа налево). Найдите сумму таких 
чисел. 

```
a = 0
for i in range(100, 999):
    if str(i) == str(i)[::-1]:
        a += i
print(a)
```
## Задание 4
В программу вводится два натуральных числа: N и K. Выведите на экран последовательность, состоящую 
из N чисел с разностью K (последовательность начинается с единицы). 
Пример ввода: 
5 
3 
Пример вывода: 
1 4 7 10 13

```
N = int(input())
K = int(input())
for i in range(N):
    print(1 + i * K, '')
```
## Задание 5
В программу вводится одно натуральное число N. Выведите на экран все значения функции f(x) = x*x + 
5x - 7 в целых точках на отрезке [0; N]. 
Пример ввода: 
10 
Пример вывода: 
-7 -1 7 17 29 43 59 77 97 119 143

```
N = int(input())
x = 0
for i in range(x, N + 1):
    f = x * x + 5 * x - 7
    x += 1
    print(f'f({x}) = {f}')
```

## Задание 9
В программу вводится число N. Программа должна вывести два столбца чисел. В первом столбце – 
числа от 1 до N, во втором – числа от N до 1. 
Пример: 
5 
1 5 
2 4 
3 3 
4 2 
5 1
```
N = int(input())
for i in range(1, N + 1):
    print(i, N - i + 1)
```
## Задание 10
В программу вводится число N. Требуется определить, является ли это число простым.
```
N = int(input())
if N < 2:
    print("Не простое число")
else:
    x = True
    for i in range(2, N):
        if N % i == 0:
            x = False
            break
    if x:
        print("Простое число")
    else:
        print("Не простое число")
```
## Задание 11
