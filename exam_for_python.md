## Тип 8 №7667

```
import itertools
a = 'ЕГЭ'
b = 'ЕЭ'
fr = itertools.product(a, repeat=5)
m = []
for i in fr:
    m.append(list(i))
count = 0
for y in m:
    if y[0] in b:
        count += 1
print(count)
```
> Пояснение: repeat=5 размещает "слово" длиной в 5 символов с повторением. <ins>m = []</ins> пустая строка, она нужна чтобы можно было обращаться к первому элементу(0). count это счетчик считающий кол-во возможных комбинаций. ``` if y[0] in b
count += 1 ``` Проверяет чтобы слово начиналось с гласной буквы, если да то увеличивает счетчик.
### Ответ: 162
